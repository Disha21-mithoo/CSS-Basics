1.) ABSOLUTE POSITIONING:
   a.) An element positioned at the specified coordinates relative to your screen top left corner.
   b.) Two values top and left can be used along with the position property to move an HTML element.

2.) RELATIVE POSITIONING:
   a.) RP changes the position of HTML element relative to where it normally appears.

3.) FIXED POSITIONING:
   a.) Allows you to fix the position of an element on the page, regardless of scrolling.
